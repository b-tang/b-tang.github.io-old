## Installation

If you'd like to run this website locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/b-tang/b-tang.github.io.git
    ```
2. Navigate into the directory:
    ```bash
    cd b-tang.github.io
    ```
3. Open `index.html` in your browser:
    ```bash
    open index.html
    ```
